AngularJS (commonly referred to as "Angular" or "Angular.js") is an open-source web application framework mainly maintained by Google.

In step one, we used Angular to grab individual elements from a list and displayed each one repeatedly. This is the resulting page:

Angular allows us to use that method by using a single method rather than having to code a complex function that could go out of hand.

In step one, we were able to display all of the attributes of each identity by using the mustache braces around our attribute, which is coded as {{item.itemArray}}.

Angular allows us to grab elements from a different file instead of having to hard code these data into every file that you'd like to use. This reduces our chances of redundancy and other mistakes.

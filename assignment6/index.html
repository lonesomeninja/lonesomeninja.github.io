Node.js

by Trung Tran

Node.js is an open source platform for developing server-side Web applications created by Ryan Dahl in mid-2009.

Node.js is used with a command line application.

It is known for its ability to use JavaScript (JS) very efficiently and usefully. Developers have the ability to write in JS on the command line and create new modules.

Node.js has an event-driven architecture that allows developers to create asynchronous I/O processes, which allows other processes to initiate before the previous ends.

These features allow Web applications to to input/output efficient operations as well as real-time Web apps.

For example, a user can create a simple chatting system within ten minutes (with enough knowledge) that can allow many users to send a message to a central server.

To illustrate, a developer can create a method that takes some time to complete, and during that process can create or use another one during the runtime of the previous method.
